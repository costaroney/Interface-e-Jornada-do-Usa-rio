## 📖 Sobre o Projeto

O **Lumina** é uma inteligência artificial conceitual desenvolvida para auxiliar estudantes em sua jornada de aprendizagem. O projeto foi criado para oferecer uma experiência de estudos personalizada, organizada e acessível, utilizando recursos de inteligência artificial para apoiar o planejamento, o acompanhamento do desempenho e a adaptação às necessidades individuais de cada usuário.

O sistema foi desenvolvido como projeto da disciplina **Interface e Jornada do Usuário**, aplicando conceitos de arquitetura da informação, design de interfaces, usabilidade e prototipação.

---

## 🎯 Problema

Muitos estudantes enfrentam dificuldades para organizar seus estudos, manter uma rotina produtiva e acompanhar seu desempenho acadêmico. Além disso, diferentes perfis de aprendizagem podem exigir estratégias específicas para melhorar o processo de aprendizado.

O Lumina foi idealizado para oferecer uma solução digital que auxilie os usuários na gestão dos estudos de forma simples e personalizada.

## 🤖 Inteligência Artificial

### Nome

**Lumina**

### Papel

Atuar como uma assistente virtual de estudos, auxiliando o usuário na organização da rotina de aprendizagem e no acompanhamento do seu desenvolvimento acadêmico.

### Personalidade Funcional

* Inteligente;
* Organizada;
* Adaptável;
* Inclusiva;
* Intuitiva.

### Capacidades

* Criar planos de estudo personalizados;
* Auxiliar na organização da rotina;
* Acompanhar o progresso do usuário;
* Disponibilizar um chat inteligente;
* Fornecer insights sobre desempenho;
* Adaptar a experiência às necessidades do estudante;
* Apoiar diferentes perfis de aprendizagem.

### Limitações

* Não substitui professores ou especialistas;
* Depende das informações fornecidas pelo usuário;
* Foi desenvolvida como uma IA conceitual para fins acadêmicos.
